# Sitios en React

Creacion de sitio a partir de tutoriales


## Proyecto Places

Instalar de manera global 
```
sudo npm install -g create-react-app
```

Crear proyecto 
```
create-react-app places
```

Instalar yarn
```	
npm install -g yarn
```

instalar material design 
```
npm install material-ui
```

Ejecutar yarn
```
yarn
```
